# 🎧 Audio Transcription with Whisper

This project uses OpenAI Whisper to transcribe audio files into text.

## 🚀 How to use (Kaggle)

1. Download the notebook
2. Upload it to Kaggle
3. Upload your audio file (m4a/mp3)
4. Change this line:
audio_path = “/kaggle/input/your-dataset/your-file.m4a”
5. Run all cells

## 📌 Output

The transcription will be printed in the output.
